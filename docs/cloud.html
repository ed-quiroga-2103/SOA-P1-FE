Almacenamiento en Cloud

Como la base de datos se almacena en MongoDB, tiene sentido seguir usando todas las herramientas que esta posee por lo cual se utiliza el servicio MongoDB Atlas para crear un cluster que alberga la base de datos.

Este se crea en la página de MongoDB de manera muy sencilla y gratuita, permite un almacenamiento de hasta 512 MB y en este se pueden habilitar direcciones IP puntuales, para que, principalmente el servidor donde se encuentre el API, la accese. Además de esto, presenta funcionalidades como manejo como usuarios, bases de datos, permisos y métricas de uso.
