Autentificación

Para el uso de la autentificacion de usuario se utilizó Keycloak como software que valide el login de los usuarios

Keycloak posee su propia base de datos que valida y guarda los usuarios de forma que se vuelve muy fácil la validación de esta información, si se quisiera, la información se puede replicar en la base de mongo para llevar un seguimiento de estos.

Con esto no tenemos que ingresar cada vez a Keycloak para ahcer revisión de esta infromación sino que se peude ingresar a Atlas donde se guarda la demás información del proyecto.
