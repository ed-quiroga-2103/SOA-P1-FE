Autentificación

Para el proceso de autentificación de usuario se utiliza Keycloak como servicio que valida el login y registro de los usuarios.

Keycloak posee su propia base de datos que almacena los usuarios de forma que se facilita la validación de esta información. De ser necesario, la información se puede replicar en la base de datos para llevar un seguimiento de estos como información para métricas y análisis.

Además de las funcionalidades de autentificación, Keycloak presenta la posibilidad de agregarle atributos personalizados a los usuarios, de manera que, si se necesita almacenar algún otro dato aparte de los planteados por Keycloak, la posibilidad existe.
