KaraokeApp

Organización para Desarrollo del Sistema

El proyecto se organiza con una arquitectura de tres capas:

* REST API y Base de Datos

Para esta etapa se asignaron dos desarrolladores encargados de implementar el REST API, y como este se tiene que conectar con la base de datos, se decidió incluir la preparación de ésta dentro de las responsabilidades de los mismos desarrolladores. Las tareas de esta etapa incluyen la creación de la base de datos como servicio cloud, conectarla con el API y que este posea todas las herramientas necesarias para su uso por la página, como manejo de usuarios y canciones.

* Aplicación Web

En esta etapa se asignaron dos desarrolladores encargados de la implementación total de la página web y sus funcionalidades. Adicionalmente, tendrán como responsabilidad la implementación de la autentificación y el manejo del streaming proveniente del API, para que la funcionalidad de karaoke se cumpla.
