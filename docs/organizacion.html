KaraokeApp

Organización/layout de los proyectos de código

El proyecto se organizó por medio de dos etapas:

* REST API - Bases de Datos

Para esta etapa se designaron dos desarrolladores, de esta forma en esta se desarrollará el API y como este se tiene que conectar a la base de datos y se puede probar sin tener la ágina web por medio de herramientas como Postman, se decidió unificar ambas. Las tareas de esta etapa incluyen la creación de la base, subirla al cloud, conectarla con el API y que este posea todas las herrramietnas necesarias para su uso por la página.

* Diseño Web y autentificación

En esta etapa se designaron también dos desarrolladores. Principalmente encargados delñ desarrollo total de la página web y sus funcionalidades. también la parte de autetificación es una de las tareas y el manejo del streaming proveniente de la base de datos y del API para que la fuincionalidad del karaoke se cumpla.
