REST API

Para esta, las opciones si bien pueden ser muchas ya se tenía visualizado usar javascript como lenguaje principal para el desarrollo de esta.

En terminos de levantar el server para los request se decidió tambión que se iba a utilizar express. Ambas decisiones vienen dadas por la experiencia previa con estas herramientas por lo que sería un avance empezar con estas.

De estas decisiones el proyecto siguió con express por su facilidad de habilitar diferentes tipos de request, lanzar mensajesde error y habilitar cors. Sin embargo, luego de esto se decidió migrar el proyecto a typescript que,si bien es cierto de alguna forma se sigue usando javscript, typescript es una plataforma que brinda más beneficios en terminos de herramientas y uso de la misma.

Por ello se usa typescript para esto. Además tiene muchas ventajas para usar diferentes tipos de bases de datos que se necesitaran durante el proyecto y la documentación sobre esta es bastante extensa.
