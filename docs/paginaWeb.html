Página Web

Para la página web se usará también typescript, las razones de esta decisión son las mismas que el punto anterior por la permitividad que posee, además se usará html y css para el diseño de las páginas como tal.

En terminos de otros toolkits, se barajaron opciones como Vue y bootstrap, sin embargo, la página no necesitó de muchas implementaciones que no se pudieran hacer por medio de la codificación normal que ya posee html.

Para ciertos casos se termino usando bootstrap, pero la decisión se basó principalmente en necesidades puntuales, no porque una fuera mejor que la otra.
