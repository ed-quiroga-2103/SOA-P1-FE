Página Web

Para la página web se usará también TypeScript. Las razones de esta decisión son las mismas que el punto anterior por las funcionalidades que posee. Además se utilizará React JS como framework para la construcción de la aplicación, implementando Chakra-UI como biblioteca de componentes.

En cuanto a otros frameworks, se consideraron opciones como Vue y Angular, sin embargo, el framework escogido junto con las bibliotecas disponibles resultan suficientes para el diseño de la aplicación.
